About

This is a static, client-side rebuild of AroundAbout. It uses Tailwind via CDN, vanilla JS for interactivity, and an optional Mapbox map.

  • No build step needed.
  • Content resides in assets/posts.json.
  • Likes & bookmarks use localStorage.
  • Map pins come from the same JSON (lat/lng fields).
